🚀 Release Notes

What's New

CuePoint ships new features every month, driven by feedback from working DJs. Every update is free for all subscribers.

v1.1
Latest March 2026

A major feature release focused on the full DJ experience — gear logistics, night-of tools, business intelligence, and a complete visual polish pass.

👔 Wardrobe & Gear 4 changes
New
Wardrobe Tracker
Track every suit and outfit with status — Clean & Ready, Needs Washing, At the Cleaners, or Dirty. Assign outfits to upcoming events. Click to cycle status instantly.
New
Charge Tracker
Flag any equipment item as battery-powered and track its charge status. Dashboard alert fires automatically if gear needs charging within 48 hours of an event.
New
Load-Out Checklists
Build reusable packing templates per event type. Start an active checklist before any gig, check items off as you load, track progress with a live progress bar.
Improved
Equipment — 4 tabs
Equipment section now has All Gear, Charging, Load-Out, and Repairs tabs. Charging tab shows only battery-powered items with one-tap status updates.
🎤 Day-Of Mode 3 changes
New
Overtime Tracker
Live clock watches your contracted end time. The moment you run over it shows minutes over and exact dollar amount owed based on your hourly rate.
New
Mic Check Log
Log acoustic issues, zones, and fixes during the gig. Notes save back to the venue record so you're prepared every time you play there.
New
Set Energy Log
Timestamp vibe notes with emoji ratings during the night. Full log shows up in your Post-Event Debrief for review after the gig.
💼 Business & Clients 4 changes
New
Rebooking Radar
Dashboard alerts when a past client's event anniversary is within 60 days and they haven't rebooked. Especially useful for corporate clients who book annually.
New
Review Request Nudge
Dashboard alert appears 2–5 days after an event with no review sent yet. One click jumps to Quick Texts to send the request while the experience is still fresh.
New
Upsell Prompts
When converting a lead to an event, Step 1 now shows relevant add-on suggestions based on event type — with pricing and total potential revenue on the booking.
New
Shot List
New tab in the event wizard. Pre-seeded with must-get moments by event type. Check off shots during the event, copy the full list to share with your photographer.
🎨 Design & Polish 5 changes
Improved
Login Page Redesign
Two-panel dark layout — brand story on the left, sign-in form on the right. Looks and feels like a real professional platform.
Improved
Event Detail — Full Page View
Clicking an event now opens a full-width profile page with the sidebar still visible. Big event type icon, stat cards, and a proper back button.
Improved
Grouped Sidebar Navigation
Sidebar now organizes all 20+ sections into 7 collapsible groups — Events, Clients, Music & Planning, Business, Gear & Team, Tools. Starts collapsed, auto-opens to your current section.
Improved
Dashboard Greeting Card
Dashboard header now sits inside a subtle gradient card with the date, your name, and quick action buttons.
Improved
Button Styling
All action buttons are now bold with black outlines for better readability. Delete buttons stay red. Primary action buttons stay accent blue.
🔧 Bug Fixes 3 fixes
Fixed
Event Types Not Showing
Event type cards in the new event wizard were not displaying when custom event types were previously set. Fixed.
Fixed
Day-Of Mode Crash
Day-Of Mode was crashing on load with a "selectedVenueAddress is not defined" error. Fixed.
Fixed
CSV Export Line Break
Tax Export CSV was generating a broken file due to a literal newline inside a string. Fixed.
